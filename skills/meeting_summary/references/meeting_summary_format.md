# Meeting Summary Formatting Guide

A strong meeting summary should be easy to scan and useful for follow-up.

Preferred order:
1. Meeting Summary
2. Key Discussion Points
3. Decisions Made
4. Action Items
5. Blockers or Risks
6. Owners
7. Next Steps

Rules:
- Keep language professional and direct
- Use bullet points for action items
- Do not add information that was not discussed
- If a section has no information, write "Not specified"
