# Brief Generator Examples

## Example 1

### Input
Client wants a landing page for a new product launch.
The page should collect leads through a form.
Brand colors are blue and white.
Need mobile responsive design.
First draft expected by next Friday.
Final delivery should include design and frontend implementation.
Copy content is not finalized yet.

### Output

## Project Overview
Create a landing page for a new product launch focused on lead generation.

## Goals
- Launch a product landing page
- Capture leads through a form
- Ensure mobile responsiveness

## Scope
- Landing page design
- Frontend implementation
- Lead form section

## Deliverables
- Landing page design
- Responsive frontend
- Lead capture form

## Timeline
- First draft expected by next Friday
- Final delivery date not fully specified

## Assumptions
- Brand colors remain blue and white
- Copy will be finalized later

## Risks or Constraints
- Copy content is not finalized yet


## Example 2

### Input
Need a website redesign for an existing consulting firm.
Pages required: home, about, services, contact.
Client wants a premium and minimal style.
Content migration from old site is needed.
Deadline is end of month.
No logo redesign needed.

### Output

## Project Overview
Redesign the website for an existing consulting firm with a premium and minimal style.

## Goals
- Refresh the brand's web presence
- Improve structure across core pages
- Migrate existing content

## Scope
- Redesign of home, about, services, and contact pages
- Content migration from old site
- No logo redesign

## Deliverables
- New website design
- Updated page layouts
- Migrated content

## Timeline
- Delivery expected by end of month

## Assumptions
- Existing branding will remain unchanged
- Content from the old site is usable

## Risks or Constraints
- Tight delivery timeline
- Content migration may require cleanup


## Example 3

### Input
Client wants dashboard UI for internal sales team.
Need login screen, sales overview, lead table, and filters.
Desktop first.
Charts can use dummy data initially.
Development is not included in this phase.

### Output

## Project Overview
Design a desktop-first dashboard UI for the internal sales team.

## Goals
- Provide a usable dashboard design for sales operations
- Include key overview and lead management screens

## Scope
- Login screen
- Sales overview dashboard
- Lead table
- Filtering UI
- Design only, no development

## Deliverables
- Dashboard UI designs
- Login screen design
- Lead management screen designs

## Timeline
- Not specified

## Assumptions
- Charts can use dummy data in the design phase
- This phase covers UI only

## Risks or Constraints
- No final timeline provided
- Development is excluded from scope
